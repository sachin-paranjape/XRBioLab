# XR Biology Lab Prompt

Build a complete 3D Virtual Reality Biology Lab using pure Three.js in a single HTML file.

Requirements:

- Use only procedural generation for all models, furniture, and environment elements.
- Do not rely on external 3D assets or fetched resources, to avoid fetch and asset-loading errors.
- Create a bright, modern biology lab with:
  - floor-to-ceiling windows
  - a forest view outside
  - a wood-slatted ceiling
  - modern lab benches
  - wooden stools
  - realistic sunlight and soft shadows
- Place floating 3D educational models on the benches, including:
  - cell parts such as DNA, nucleus, and mitochondria
  - human organs such as a heart, brain, lungs, and bone
- Add VR teleportation so the user can move around the space naturally.
- Make all models interactive:
  - when clicked or pointed at, show an info pane or tooltip
  - read the description aloud using native text-to-speech
- Keep the experience self-contained, lightweight, and suitable for browser-based XR exploration.

Goal:

Create an immersive educational VR biology lab that demonstrates prompt-driven XR development, procedural scene generation, and interactive learning in a single-file Three.js experience.
